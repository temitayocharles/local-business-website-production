# the-duffy Website Bundle

This is a deployable multi-page bundle. Files:

- `index.html`
- `menu.html`
- `about.html`
- `gallery.html`
- `contact.html`

## Quick Preview

```bash
python3 -m http.server 8080
```
Then open `http://localhost:8080/index.html`.

## Assets

- `assets/placeholder.svg` for gallery placeholders. Replace with real images.
